# `mlmm pysis`

```text
mlmm-toolkit ver. 0.2.5.dev18

Usage: mlmm pysis [OPTIONS] [YAML_FILE]

  Run a pysisyphus YAML workflow file.

  This subcommand provides compatibility with v0.1.x YAML-based workflows. The
  ``mlmm`` calculator type is automatically registered so that ``calc: type:
  mlmm`` works in pysisyphus YAML files.

  Usage:
      mlmm pysis opt.yaml
      mlmm pysis tsopt.yaml -- --clean

Options:
  --help-advanced  Show all options (including advanced settings) and exit.
  -h, --help       Show this message and exit.
```
